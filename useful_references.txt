random - Module for generating random numbers and selecting random elements
Example:
       number = random.randint(1,100)


Comparisons - Operations for comparing values
Example:
       x < y


f-strings - Strings into which expressions can be interpolated
Example:
       f'It is currently{datetime.datetime.now()}'


for loops - Iterates over the elements of an iiterable
Example:
       for i in range(10):
           print(i*i)


enumerate - Helps us to number elements of iterables
Example:
       for index, item in enumerate('abc')
           print(f'{index}: {item}')
